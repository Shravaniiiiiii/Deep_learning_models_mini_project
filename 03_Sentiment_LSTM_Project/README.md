# Sentiment Analysis using LSTM

This project performs sentiment analysis on movie reviews using an LSTM deep learning model.

## Dataset
IMDB Movie Reviews Dataset (built-in TensorFlow dataset)

## Model Used
- Embedding Layer
- LSTM Layer
- Dense Output Layer

## Accuracy
Achieved ~85% accuracy on test data.

```bash
git clone https://github.com/yourusername/Sentiment_LSTM_Project.git
cd Sentiment_LSTM_Project
pip install -r requirements.txt
python sentiment_lstm.py
